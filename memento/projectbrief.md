# Project Brief: AI/ML Portfolio Website

## Owner
**Kotari Chaitanya Krishna** — AI/ML Engineer

## Goal
Build a modern, professional, and highly attractive AI/ML portfolio website with animations, responsive design, and GitHub Pages deployment.

## Core Requirements
- Fully responsive (mobile + desktop)
- Dark theme with glassmorphism UI
- Animated hero section with typing effect
- Scroll-based animations (Framer Motion)
- Interactive skill bars
- Contact form with EmailJS validation
- Download Resume button
- Particle background animation
- Scroll progress indicator
- Loading animation

## Tech Stack
- React.js (Vite)
- Tailwind CSS
- Framer Motion
- EmailJS
- tsParticles
- GitHub Pages deployment

## Deployment Target
https://github.com/Chaitanya8822/Portfolio.git → GitHub Pages
